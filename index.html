Welcome to IceFireDB
IceFireDB is a database built for web3 and web2,The core mission is to help applications quickly achieve decentralization.
The storage layer supports disk,OSS,IPFS and other storage methods,
the protocol layer currently supports RESP, and will support SQL and GraphQL in the future.
A blockchain fusion layer based on Ethereum and EOS is being built,
which can be used in fusion with higher-level decentralized computing platforms and applications as well as identity,
financial assets, intellectual property, and sidechain protocols.
IceFireDB strives to fill the gap of the decentralized stack, making web3 application data storage more convenient,
and web2 application easier to achieve decentralization.

What is IceFireDB?

IceFireDB is a database built for web3 and web2, and strives to fill the gap between web2 and web3 with the help of a friendly database experience, making web3 application data storage more convenient, and making web2 applications easier to access the blockchain.

IceFireDB helps build NoSQL on IPFS

The bottom layer of IceFireDB supports the IPFS decentralized storage driver, and the middle layer supports a complete NoSQL encoder, helping IPFS storage to have more data types.

Web3 Dao community organized by IceFireDB

In order to better build and motivate community members, IceFireDB is cooperating with weelink web3 dao platform to promote the construction of cloud native Dao community.
